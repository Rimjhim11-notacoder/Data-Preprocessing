# Data-Preprocessing
This repo has sample codes for various preprocessing methods

Preprocessing is done on raw data to ready it for further data analysis techniques and also improve its quality. 
Steps include:
1. Data Cleaning (Missing Values, Noisy Data, Outlier Analysis, Inconsitent Data(values collected in different metric(inch - cm ))
2. Data Transformation and Aggregation(Data Normalisation(or scaling) , aggregation(weekly, monthly, yearly) and Integration(from multiple sources)
3. Data Reduction (Reducing Data Volume, Data Sampling, Reducing Data Attributes, Attributes Subset Selection)
4. Proximity Analysis (for : Nominal Attributes, Binary Atttributes, Numerical Attributes, Ordinal & Mixed Attributes)
